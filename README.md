# Las Vegas Pool House

Luxury vacation rental booking website for Las Vegas Pool House.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment:
```bash
cp .env.example .env.local
# Edit .env.local with your Beds24 property ID
```

3. Run development server:
```bash
npm run dev
```

## Deployment

```bash
vercel --prod
```

## Tech Stack
- Next.js 14
- Tailwind CSS
- Beds24 API
- Vercel hosting

See [SPEC.md](./SPEC.md) for full documentation.
